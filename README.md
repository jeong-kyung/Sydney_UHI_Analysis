# Sydney_UHI_Analysis
This project analyzes the correlation between NDVI and LST in Sydney to propose urban cooling strategies based on biological mechanisms.
----------
시드니 녹지 지수(NDVI)와 지표 온도(LST) 간의 상관관계 및 온도 저감 효과 분석 프로젝트
----------
## 🧬 핵심 분석 및 생명공학적 해석(Key Insight)
Confirmed strong negative correlation (-0.79) and the cooling effect of latent heat through plant evapotranspiration.
QGIS-based Data Preprocessing**: Extracted grid-specific NDVI and LST values using spatial data analysis to ensure spatial accuracy.
* **Statistical Significance**: Confirmed a strong negative correlation (**-0.79**) between greenery density (NDVI) and surface temperature (LST).
* **Biological Insight**: Quantitatively proved that the **Latent Heat** conversion mechanism, driven by plant **Evapotranspiration**, is the key driver in reducing urban surface temperatures.
-----------
- **QGIS 기반 데이터 전처리**: 공간 데이터를 활용한 격자별 NDVI, LST 수치 추출
- **통계적 유의성 확보**: NDVI와 LST 간의 강한 음의 상관관계(-0.79) 확인
- **Biological Insight**: 식물의 증산작용(Evapotranspiration)을 통한 잠열(Latent Heat) 전환 메커니즘이 도심 지표 온도 저감의 핵심 동력임을 정량적으로 입증하였습니다.

## 🛠 Tech Stack

- **Libraries**: Pandas, Seaborn, Matplotlib

# Sydney_UHI_Analysis
시드니 녹지 지수(NDVI)와 지표 온도(LST) 간의 상관관계 및 온도 저감 효과 분석 프로젝트
## 🧬 핵심 분석 및 생명공학적 해석
- QGIS 기반 데이터 전처리: 공간 데이터를 활용한 격자별 NDVI, LST 수치 추출
- 통계적 유의성 확보: NDVI와 LST 간의 강한 음의 상관관계(-0.79) 확인
- Biological Insight: 식물의 증산작용(Evapotranspiration)을 통한 잠열(Latent Heat) 전환 메커니즘이 도심 지표 온도 저감의 핵심 동력임을 정량적으로 입증하였습니다.

## 🛠 Tech Stack
- Tools: QGIS, Google Colab
- Libraries: Pandas, Seaborn, Matplotlib
